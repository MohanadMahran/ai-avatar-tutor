Extraction Details & Quality Safeguards
Spatial Reconstruction: To avoid text-box positioning artifacts in the PDF streams (such as out-of-order headers or code segments), we group visual words by vertical lines and sort them.
Indentation Recovery: We will calculate code indentations dynamically from word start coordinates (x0) using the formula spaces = max(0, round((x0 - 63.75) / 6.625)).
Formatting Cleanup: We strip page headers/footers, markdown fences (e.g. ````python), and programming language headers (e.g. python, css`) that were added visually in the PDF but are not part of the file contents.
